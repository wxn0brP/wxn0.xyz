# wxn0.xyz

Source for the hacking mini-game at [wxn0.xyz](https://wxn0.xyz).

## How to Play

This is a small hacking-themed game where you gain XP, level up, and unlock links to other projects. Type commands into the terminal to play.

### Commands

-   `help`: Shows the list of available commands.
-   `status`: Displays your current level and XP.
-   `hack`: Starts a new hacking mission to gain XP.
-   `links`: Shows all the links you have unlocked.
-   `clear`: Clears the terminal screen.
-   `reset`: Resets all your game progress.
-   `welcome`: Displays the welcome message again.

## Development

This is a TypeScript project built with [Bun](https://bun.sh/).

### Setup

Install dependencies:
```bash
bun install
```

### Build

To build the project, run:
```bash
bun build
```
This will compile the TypeScript files from `src/` into JavaScript in the `dist/` directory.
